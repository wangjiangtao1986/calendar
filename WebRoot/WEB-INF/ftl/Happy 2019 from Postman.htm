Postman Monthly Newsletter: January 2019
 

Happy 2019 from Postman!

Postman made huge strides in 2018!

In 2018, we saw our user base grow from 3 million to 6 million developers and from 100,000 to more than 200,000 companies. We launched numerous new features, such as Workspaces, free teams, and Open API 3.0 support. We released Postman Enterprise for larger teams with enterprise-level security and governance needs. And, we grew our own team to speed up our product development with the goal of making API development easier for everyone. 

See what we have in store for you in 2019 on our Public Postman Roadmap. Thank you for an amazing 2018. We can't wait to see where this year takes us!
See our roadmap
 
 
 

Features of the Month

  • We released forking and merging in Postman! This new version control functionality makes it easier than ever for teams to collaborate on API development within Postman.
  • We added an interactive in-app learning center where you can explore and master Postman skills that address every stage of the API lifecycle from designing and mocking, to testing and documenting.
  • If you use Git, we think you'll enjoy our extended GitHub and GitLab integration capabilities. Now Postman Pro and Enterprise users can back up collections using custom domains.
